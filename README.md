# Face Recognition App

## 🔍 Overview
This project detects faces from images and identifies known persons using Python.

## ⚙️ Tech Stack
- Python
- OpenCV
- face_recognition
- Streamlit

## 🚀 Features
- Detects faces in uploaded images
- Matches with known faces
- Shows name or "Unknown" if not found

## ▶️ How to Run
```bash
pip install -r requirements.txt
streamlit run app.py
